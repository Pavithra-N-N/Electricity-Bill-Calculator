# ⚡ Electricity Bill Calculator

## 📌 Project Description

The Electricity Bill Calculator is a simple web application developed using HTML, CSS, and JavaScript. It calculates the electricity bill based on the number of units consumed and displays a detailed bill summary.

---

## ✨ Features

- Calculate electricity bill
- Customer name input
- Unit-based billing calculation
- Automatic bill summary
- Reset form
- Attractive and responsive user interface
- Input validation

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript

---

## 📚 Concepts Used

- Variables
- Functions
- If-Else Conditions
- DOM Manipulation
- Event Handling
- User Input

---

## 💰 Tariff Used

| Units | Rate |
|-------|------|
| 0 – 100 | ₹1.50 / unit |
| 101 – 200 | ₹2.50 / unit |
| 201 – 300 | ₹4.00 / unit |
| Above 300 | ₹6.00 / unit |

---

## ▶️ How to Run

1. Download the project.
2. Open the folder in VS Code.
3. Open `index.html` in a browser or use Live Server.
4. Enter the customer name and units.
5. Click **Calculate Bill**.
